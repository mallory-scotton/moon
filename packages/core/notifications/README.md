# @moon/notifications

Push notifications, webhooks, alerts for Moon HTPC

Please refer to the [documentation](./docs) for more information.

## Installation

```sh
yarn add @moon/notifications
# or
npm i @moon/notifications
# or
pnpm i @moon/notifications
```

# @moon/search

Multi-indexer search engine (torrents, usenet) for Moon HTPC

Please refer to the [documentation](./docs) for more information.

## Installation

```sh
yarn add @moon/search
# or
npm i @moon/search
# or
pnpm i @moon/search
```

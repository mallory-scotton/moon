# @moon/ffmpeg

FFmpeg integration for Moon HTPC

Please refer to the [documentation](./docs) for more information.

## Installation

```sh
yarn add @moon/ffmpeg
# or
npm i @moon/ffmpeg
# or
pnpm i @moon/ffmpeg
```

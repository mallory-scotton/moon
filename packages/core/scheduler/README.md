# @moon/scheduler

Cron jobs, background tasks and automation for the Moon HTPC

Please refer to the [documentation](./docs) for more information.

## Installation

```sh
yarn add @moon/scheduler
# or
npm i @moon/scheduler
# or
pnpm i @moon/scheduler
```
